adoptada?
Solución 1 - git pull + resolver conflictos + git push:
  Cada uno hace git pull origin TESTING, resuelve los conflictos 
  manualmente en index.html, y luego hace git push. 
  Consecuencia: se conservan todos los cambios de todos, pero hay 
  que resolver conflictos a mano.

H- ¿Qué sucede si cada uno utiliza el push -f?
El último en ejecutar el push -f gana y sobreescribe todo lo anterior.
Es decir, en el MASTER solo quedaría el <p> de GAMMA, perdiéndose 
los <p> de ALFA y BETA completamente.
